    aspect-ratio

    Since 1.0.0
    @mixin aspect-ratio($ratio: 1) { ... }

    Description

    Generate declarations in order to set the ratio width/height equals to the argument passed.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $ratio—noneNumbers or List1

    Output

    ::before and ::after elements css declarations

    Example

    .class { @include aspect-ratio(); }            // no arguments defaults to a 1:1 ratio
.class { @include aspect-ratio(16, 9); }       // width and height
.class { @include aspect-ratio(1.777778); }    // ratio (calculated width/height)
.class { @include aspect-ratio(4px, 3px); }    // comparable units
.class { @include aspect-ratio($ratio: 1.2); } // keywords

    • The original codepen it comes from

    gutter-row

    Since 1.3.0
    @mixin gutter-row($gutter: 30px, $vertical: false) { ... }

    Description

    Generate margin on left & right side for the row of a grid in order to setup a gutter

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $gutter

    the gutter space expected into your grid

    		Numbers30px
    $vertical

    Bool to know if margin-bottom will be generated for vertical gutter-space

    		Booleansfalse

    Output

    margin-left, margin-right & maybe margin-bottom declarations

    Example

    .row {
  @include gutter-row(25px);
}

.row {
  @include gutter-row; // default value is `30px`
}

.row {
  @include gutter-row(25px, true); // for grid with vertical gutter space
}

    Author

    • Loïc Goyet

    Generate padding or margin on left & right side for the col of a grid in order to setup a gutter

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $gutter

    the gutter space expected into your grid

    		Numbers30px
    $-bottom`

    will be generated for vertical gutter-space

    		Booleans} $vertical [false] - bool to know if `{margin or Padding—none
    $property

    the properties' declaration generated (can only be 'padding' or 'margin')

    		String'padding'

    Output

    margin-left & margin-right or padding-left & padding-right declarations

    Example

    .col {
  @include gutter-col(25px);
}

  @include gutter-col; // default value is `30px`
}

  @include gutter-col(25px, true); // to generate `padding-bottom` declarations for vertical gutter-space

    get-hypotenuse

    Since 1.2.0
    @function get-hypotenuse($ab, $ac) { ... }

    Description

    Get the hypotenuse length based on the Pythagoras' theorem. Reminder : ab² + ac² = bc² on a right-angled triangle. We pass as argument ab and ac length, and the function returns bc length

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $ab

    first side of the right-angled

    		Numbers—none
    $ac

    second side of the right-angled

    		Numbers—none

    Returns

    Number —

    the hypothenuse length

    Throws

    • The two value into the get-hypotenuse() function must share the same unit type

    Requires

    • [function] square
    • [function] square-root

    • Loïc Goyet

    block-generator

    Since 1.4.0
    @mixin block-generator($declarations) { ... }

    Description

    Generate list of declarations

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $declarations—noneMap—none

    Output

    one or several css declarations

    Example

    $pod--declarations: (
  'margin': 1em 0,
  'padding': 1em,
  'background-color': rgb(255, 0, 0),
);

.pod {
  @include block-generator($pod--declarations);
}

// compiles to
.pod {
  margin: 1em 0;
  padding: 1em;
  background-color: rgb(255, 0, 0);
}
